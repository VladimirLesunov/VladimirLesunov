Senior Frontend Engineer with 8+ years building cross-platform apps at scale. Previously **Desktop Team Lead** at **Termius**, where I led 11 engineers shipping SSH & terminal tools across macOS, Windows, Linux, iOS, and Android.

I focus on **performance**, **release safety**, and **developer experience** — from migrating 5,700+ files to Vite (1 min → 12 sec startup) to building gradual rollout pipelines with Terraform and AWS.

Currently exploring new opportunities in frontend engineering and technical leadership.
